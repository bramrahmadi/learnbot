## ðŸ”§ Troubleshooting

### Common Issues

| Issue | Solution |
|-------|---------|
| Resume upload fails | Check file format (PDF/DOCX only) and size (max 10MB) |
| Skills not recognized | Add skills manually with exact industry terminology |
| Low match score for a job I'm qualified for | Ensure all relevant skills are in your profile |
| Session expired | Log in again â€” sessions last 24 hours |
| Page not loading | Clear browser cache and refresh |
| Job search returns no results | Remove some filters and try a broader search |

### Error Messages

| Error | Meaning | Solution |
|-------|---------|---------|
| "Session expired" | Your 24-hour session has ended | Log in again |
| "File too large" | Resume exceeds 10MB | Compress or reduce file size |
| "Unsupported file type" | Wrong file format | Use PDF or DOCX |
| "Parse error" | Resume couldn't be read | Check formatting tips |
| "Too many requests" | Rate limit exceeded | Wait a moment and try again |

2. **Use standard section headings** â€” "Experience", "Education", "Skills"
3. **Avoid tables and text boxes** â€” These confuse the parser
4. **Use text-based PDFs** â€” Scanned images cannot be parsed
